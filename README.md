# Solow-Swan-model-with-R-D-Expenditure
The project features an implementation and econometric analysis of the Macroeconomic Solow-Swan Growth Model, with enhancements by including Research and Developments expenditure as a determinant. The OLS regression is used to analyze relationship between GDP per capita, savings rate, population growth and R&D expenditure across multiple countries over time.

The data is from the World Bank and th eproject follow the methodology outlined bt Mankiw et al.(1992), including data cleaning techniques and statistical testing to ensure robustness.
